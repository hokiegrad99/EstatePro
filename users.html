Platform Admin

Platform-level concerns that aren't part of any single estate live here.

  • Register a User — create a Firebase Auth account so someone can sign in even without an invite.
  • Reset Password — reset another user's Firebase Auth password.
  • Delete User — remove a user's account from the platform entirely.
  • Export Accounts CSV — audit-friendly listing of everyone with platform access.

Per-estate membership (roles, invitations, member removal) lives on Executor. The role dropdowns that used to be on this page were confusing post-Phase 4 — the per-estate model owns those.
