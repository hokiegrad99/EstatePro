Founder Bootstrap

EstatePro — create the very first estate

You must sign in or register first.
Your Firebase project already has one or more estates. The founder bootstrap path is only for the very first estate. Sign in and ask an existing executor to invite you instead.

Paste the founder secret you generated when you set up Firebase (see FIREBASE-SETUP.md). The secret is never transmitted in plaintext — it’s SHA-256 hashed in your browser before the request reaches Firestore.

This is a long random string (e.g., 64 hex chars). After this bootstrap you can delete it from your offline notes — the SHA-256 inside firestore.rules is what gates create access.
